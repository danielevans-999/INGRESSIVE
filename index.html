About the
Scholarship

According to a report by Cybersecurity Ventures, the number of unfilled cybersecurity jobs grew from 1 million positions in 2013 to 3.5 million in 2021. 

A lot of these roles are still yet to be filled because of a lack of talent. Yet, predictions show that there would be the same number of openings in 2025. 

This is why we have partnered with Cisco to train African youths and help them build careers in Cybersecurity 

What you will Learn

You will learn the following:

  • Introduction to Cybersecurity
  • Cybersecurity Essentials Management
  • Cyber Threat
  • Security
  • Defence
  • Endpoint Security
